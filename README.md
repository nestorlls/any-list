## Stack used

- Nest
- GraphQl
- Docker
- PostgreSQL

## Dev Installation

1. Clone repository
2. Copy the file `.env.template` and rename to `.env`
3. Install dependencies

```bash
$ pnpm install
# or
$ yarn install
# or
$ npm install
```

4. Keep up docker image data base (Docker Desktop needed)

```bash
$ docker compose up -d
```

5. Run the Nest server

```bash
$ pnpm start:dev
```

6. Go to Browser and type `http://localhost:3000/graphql`
